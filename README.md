# Text-Transcription
Node js application, which enables the user to extract transcriptions from audio/video file.

# Libraries
  axios,
  cookie-parser,
  cors,
  dotenv,
  ejs,
  express,
  express-fileupload,
  google-auth-library,
  mongoose

# API
To extract transcriptions from audio/video file, I have used AssemblyAI API.
https://www.assemblyai.com/
